## Task: Implement Home screen UI skeleton

**Goal**

Create the basic Home screen layout that will serve as the post-login landing page, ready to host the campaigns list.

**Description**

- Create a `HomeScreen` route file (e.g. `app/home.tsx` or replace the default `(tabs)/index.tsx` content) following Expo Router conventions.
- Use the shared `Screen` layout component as the root wrapper.
- Add a title section (e.g. app name and/or \"Your Campaigns\").
- Add placeholder space or components where the campaigns list will render.

**Acceptance Criteria**

- The Home screen route exists and can be navigated to from app launch.
- Visual layout includes a clear title and content area, even before the campaigns list is wired up.
- Home screen uses the `Screen` layout component and themed primitives.

**Notes**

- Keep this layout simple and focused; detailed styling and the actual list rendering are covered in subsequent tasks.
